Now, when i reduce the size of the browser window,I noticed that the cities row breaks first. This happens because we specified the breakpoint for the cities row using col-md, which means that on small screens (768px and below), the letters will start stacking vertically. The letter names, specified with col-sm-2, will continue to display horizontally until the screen becomes even narrower.
